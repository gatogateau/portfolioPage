A group interactive game that pits friends against each other. Each player signs up, joins a game and is assigned a target. The assassin must eliminate the target by using the camera on his/her phone. Once the target is eliminated, the assassin receives the target's target. This continues until the last target is eliminated.

The game showcases the use of ReactJS and passport authentication and facial recognition on the client side. The server side uses Node JS, Mongo DB, mongoose, Express. The heavy lifting of this application is done on the server side, therefore keeping the site running efficiently and secure.
