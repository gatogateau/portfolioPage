Photo-Ops

A group interactive game placing friends against each other in a competitve setting. Each player signs up, joins a game and is assigned a target. The assassin (user) must eliminate the target by using the camera on his/her phone to take a picture of their target. Once the target is eliminated, the assassin receives their target's target. This continues until the last target is eliminated.

This game showcases the use of ReactJS, passport authentication and facial recognition on the client side. The server side uses Node JS, Mongo DB, Mongoose, and Express. The heavy lifting of this application is done on the server side, where the majority of the logic is done in the controllers' pages, and the API requests through the routes folders; therefore keeping the site running both secure and efficiently.
